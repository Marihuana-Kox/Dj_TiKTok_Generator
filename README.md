# 🤖 AI Content Generator Platform
ВАЖНО - ПРОГРАММ ПОКА В РАЗРАБОТКЕ И ФУНКЦИОНАЛ НЕ ПОЛНЫЙ

Платформа для автоматизированной генерации контента с использованием искусственного интеллекта. Система позволяет создавать статьи, изображения, озвучку и видеоматериалы на основе заданных тем и идей.

## 📋 Описание

Это Django-приложение, которое предоставляет полный цикл создания контента:

1. **Генерация идей** — создание концепций для видео/статей с различными стилями подачи
2. **Написание статей** — автоматическое создание структурированного контента на нескольких языках
3. **Генерация изображений** — создание иллюстраций через AI (Flux, Stable Diffusion)
4. **Синтез речи** — озвучка текста через ElevenLabs
5. **Монтаж видео** — автоматическая сборка финального видеоматериала

### Основные модули

| Модуль | Описание |
|--------|----------|
| `topics` | Управление идеями и проектами (VideoProject) |
| `article` | Создание и перевод статей, кластеризация контента |
| `image` | Генерация промптов и изображений для контента |
| `generator` | Управление медиа-активами и сценариями |
| `ai_inspector` | Конфигурация AI-провайдеров (OpenAI, Gemini, HuggingFace) |
| `config` | Глобальные настройки системы |
| `prompts` | Библиотека шаблонов промптов для разных задач |

## ✨ Особенности

### 🔧 Два режима работы
- **Ручной режим** — пошаговый контроль каждого этапа генерации
- **Автопилот** — полностью автоматический процесс (One-Click)

### 🌐 Мультиязычность
- Поддержка нескольких языков (RU, DE, EN и другие)
- Автоматический перевод статей
- Настраиваемые языковые пакеты

### 🎨 Гибкая система промптов
- Библиотека шаблонов для идей, структур и статей
- Версионирование промптов
- Возможность кастомизации под разные стили

### 🔐 Безопасное хранение ключей
- Шифрование API-ключей провайдеров
- Поддержка множества AI-сервисов

### 🖼️ Генерация изображений
- Поддержка Flux Schnell, Flux Dev, Stable Diffusion XL
- Настройка соотношения сторон (9:16, 16:9, 1:1, 4:3)
- Привязка изображений к сценам статьи

## 🚀 Установка

### Требования

- Python 3.10+
- SQLite (по умолчанию) или PostgreSQL
- API-ключи для выбранных AI-сервисов

### Шаг 1: Клонирование репозитория

```bash
git clone <repository-url>
cd <project-directory>
```

### Шаг 2: Создание виртуального окружения

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# или
venv\Scripts\activate  # Windows
```

### Шаг 3: Установка зависимостей

```bash
pip install -r requirements.txt
```

### Шаг 4: Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```bash
# Ключ шифрования для AI-ключей (сгенерируйте новый!)
AI_ENCRYPTION_KEY=your-encryption-key-here

# Или используйте существующие ключи напрямую в админке Django
```

Для генерации ключа шифрования:
```python
from cryptography.fernet import Fernet
print(Fernet.generate_key().decode())
```

### Шаг 5: Миграции базы данных

```bash
python manage.py migrate
```

### Шаг 6: Создание суперпользователя

```bash
python manage.py createsuperuser
```

### Шаг 7: Запуск сервера разработки

```bash
python manage.py runserver
```

Приложение будет доступно по адресу: http://127.0.0.1:8000/

### Шаг 8: Настройка через админ-панель

1. Войдите в админ-панель: http://127.0.0.1:8000/admin/
2. Настройте AI-провайдеров в разделе **"Провайдеры ИИ"**
3. Добавьте API-ключи:
   - OpenAI (для генерации текста)
   - Google Gemini (альтернатива для текста)
   - HuggingFace (для генерации изображений)
   - ElevenLabs (для озвучки)
4. Настройте глобальные параметры в **"Настройки системы"**
5. Создайте шаблоны промптов в соответствующих разделах

## ⚙️ Конфигурация

### Настройка AI-провайдеров

В админ-панели перейдите в раздел **ai_inspector → Провайдеры ИИ** и добавьте:

| Провайдер | Тип | Назначение |
|-----------|-----|------------|
| `openai` | LLM | Генерация статей, переводы |
| `gemini` | LLM | Альтернативная текстовая модель |
| `hf` | Image | Генерация изображений |
| `elevenlabs` | Audio | Синтез речи |

### Глобальные настройки

В разделе **config → Настройки системы**:

- **Режим работы**: ручной или автопилот
- **API-ключи**: прямое указание ключей (не рекомендуется для продакшена)
- **Модели по умолчанию**:
  - Для статей: GPT-4o, GPT-4o Mini, Gemini 1.5 Flash/Pro
  - Для изображений: Flux Schnell, Flux Dev, SDXL
- **Авто-настройки**:
  - Языки для автоперевода
  - Автогенерация изображений
  - Автогенерация озвучки
  - Автомонтаж видео

### Шаблоны промптов

#### 1. Промпты для идей (`prompts`)
Определяют стиль подачи материала:
- Факты и вопросы
- Сенсационный/кликбейт
- Загадки и тайны
- Образовательный

#### 2. Промпты для структуры
Задают скелет статьи: Hook → Body → Climax → Conclusion

#### 3. Промпты для статей
Шаблоны для написания полного текста

## 💡 Использование

### Создание нового проекта

1. В админ-панели создайте новую запись в **topics → Идеи**
2. Укажите:
   - Тему видео
   - Ракурс/парадокс (основная идея)
   - Заметки и факты
3. Выберите статус и запустите генерацию

### Работа в режиме автопилота

При включенном режиме `auto`:
1. Система автоматически сгенерирует структуру статьи
2. Напишет полный текст на основном языке
3. Переведёт на указанные языки
4. Сгенерирует промпты для изображений
5. Создаст изображения
6. (Опционально) создаст озвучку и смонтирует видео

### Ручной режим

Каждый этап требует подтверждения:
1. ✅ Генерация идеи → подтверждение
2. ✅ Создание структуры → редактирование → подтверждение
3. ✅ Написание статьи → редактирование → подтверждение
4. ✅ Перевод → проверка → подтверждение
5. ✅ Генерация изображений → выбор лучших вариантов

## 🔌 API-провайдеры

### Поддерживаемые сервисы

- **OpenAI** — GPT-4o, GPT-4o Mini
- **Google Gemini** — 1.5 Flash, 1.5 Pro
- **HuggingFace** — Flux Schnell, Flux Dev, SDXL
- **ElevenLabs** — синтез речи

### Добавление нового провайдера

1. Откройте админ-панель
2. Перейдите в **ai_inspector → Провайдеры ИИ**
3. Нажмите "Добавить"
4. Заполните:
   - Кодовое имя (например, `anthropic`)
   - Отображаемое имя
   - Тип (llm/image/audio/video)
   - API-ключ (будет зашифрован)
   - Base URL (если отличается от стандартного)
   - Дополнительная конфигурация (JSON)

## 🗄️ Модель данных

### Ключевые модели

- **VideoProject** — основной проект (идея видео/статьи)
- **ArticleCluster** — группа статей на одну тему (для разных языков)
- **ArticleTranslation** — перевод статьи на конкретный язык
- **ImagePrompt** — промпт для генерации изображения
- **MediaAsset** — медиафайл (изображение, аудио, скрипт)
- **ScriptData** — данные сценария (текст, переводы, хэштеги)
- **AIProvider** — конфигурация AI-сервиса
- **SystemConfig** — глобальные настройки

## 📝 Переменные промптов

В шаблонах промптов поддерживаются переменные:

- `{topic}` — тема материала
- `{angle}` — ракурс/парадокс
- `{notes}` — заметки и факты
- `{language}` — целевой язык
- `{context}` — дополнительный контекст

## ⚠️ Важные замечания

1. **Безопасность ключей**: В продакшене храните `AI_ENCRYPTION_KEY` только в переменных окружения
2. **Лицензии API**: Убедитесь, что у вас есть права на использование выбранных AI-сервисов
3. **Лимиты запросов**: Некоторые провайдеры имеют ограничения на количество запросов
4. **Стоимость**: Генерация контента может быть платной в зависимости от выбранных моделей
